# v1.8.9 APK New Sale Acceptance Criteria

## Amac

APK build oncesi Yeni Fis ekraninin kabul kriterlerini netlestirmek.

## Kabul kriterleri

- Yeni Fis ekrani beyaz ekran vermeden acilir.
- Eski taslak sifirlanabilir.
- Musteri secimi calisir.
- Fis baslatma calisir.
- Barkod veya demo kod ile urun bulunur.
- Fiyat gorunur.
- Adet direkt girilebilir.
- Urun adet ve fiyatla fise eklenir.
- Fis toplam tutari gorunur.
- QR Album fiyat gostermez.

## APK karari

Bu kriterler gecmeden APK test fazi tamamlanmis sayilmaz.
