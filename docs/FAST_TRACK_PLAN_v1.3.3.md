# v1.3.3 Fast Track Plan

## Amac

Terminal uygulamasinda gelistirme hizini artirmak ve paketleri daha genis kapsamli ilerletmek.

## Yeni calisma modeli

- 1 paket 6-8 kucuk/orta adimdan olusur.
- Dusuk riskli UI, mock, dokuman ve config degisiklikleri GitHub uzerinden ilerletilir.
- Buyuk refactor, local build, APK build ve cihaz testi gereken konular ayrica isaretlenir.
- Her paket sonunda Web Preview kontrol linki verilir.
- Her paket sonunda yuzdesel durum raporu eklenir.

## Oncelik sirasi

1. Web Preview demo akisini guclendirmek
2. APK hazirlik ve Honeywell test planini netlestirmek
3. ERP entegrasyonu oncesi risk sinirlarini yazmak
4. Canli kullanim oncesi eksikleri ayirmak
