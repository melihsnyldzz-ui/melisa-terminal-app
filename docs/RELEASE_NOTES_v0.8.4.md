# v0.8.4 Release Notes

## Paket hedefi

Bu paket, web preview ile test edilebilecek operasyon ekranlarını güçlendirmek için hazırlandı.

## Gelen değişiklikler

- Mesajlar ekranına mesaj, fiş no ve gönderen araması eklendi.
- Mesajlar ekranına görünen mesaj, fişe bağlı mesaj ve acil mesaj özet kartları eklendi.
- Mesaj bulunamadığında daha net boş durum metni gösterilir.
- GitHub Pages için Expo web base path ayarı korunmuştur.

## Sürüm

- package.json version: 0.8.4
- app.json expo.version: 0.8.4

## Not

Bu paket GitHub üzerinden uygulanmıştır. Web Preview workflow başarılı tamamlandığında değişiklikler GitHub Pages sitesine otomatik yansır.
