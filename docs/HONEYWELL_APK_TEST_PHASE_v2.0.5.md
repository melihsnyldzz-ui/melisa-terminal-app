# v2.0.5 Honeywell APK Test Phase Plan

## Amac

Web Preview tarafinda demo akis dogrulandiktan sonra Honeywell ve APK test fazina gecis siralamasini netlestirmek.

## Test fazi sirasi

1. Web Preview akisi kontrol edilir.
2. Yeni Fis fiyat ve adet akisi tamamlanir.
3. QR Album fiyat gostermeden kontrol edilir.
4. Local npm run check calistirilir.
5. Expo preview ile ekranlar kontrol edilir.
6. Honeywell cihazda barkod focus testi yapilir.
7. APK build karari verilir.

## Kritik karar

Honeywell cihazda barkod okutma ve adet girme davranisi gecmeden canli kullanima gecilmez.
