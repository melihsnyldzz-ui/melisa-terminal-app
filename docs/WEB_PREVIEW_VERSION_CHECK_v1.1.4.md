# v1.1.4 Web Preview Version Check

## Amac

Web Preview sayfasinda dogru surumun gorundugunu kontrol etmek.

## Kontrol adimlari

1. Web Preview linkini gizli sekmede ac.
2. Login ekranindaki surum etiketini kontrol et.
3. GitHub Actions bolumunde Web Preview workflow sonucunu kontrol et.
4. Workflow yesil tamamlanmadan eski surum gorunebilir.
5. Eski surum gorunurse cache temizle veya versiyon parametresiyle tekrar ac.

## Beklenen sonuc

Web Preview sayfasinda uygulama surumu, GitHub uzerindeki guncel uygulama surumuyle uyumlu gorunmelidir.
