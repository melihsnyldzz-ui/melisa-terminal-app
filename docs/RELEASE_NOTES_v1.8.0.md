# v1.8.0 Release Notes

## Paket hedefi

Yeni Fis beyaz ekran riskini azaltmak ve local storage okuma katmanini guclendirmek.

## Degisiklikler

- Local storage okuma islemi try/catch ile guvenli hale getirildi.
- Bozuk JSON kaydi varsa uygulama cokmeden kayit temizlenir.
- Aktif satis taslagi yuklenirken fiyat ve adet degerleri normalize edilir.
- Yeni Fis beyaz ekran recovery notu eklendi.
- Paket surumu 1.8.0 olarak hizalandi.

## Not

Hala beyaz ekran gorulurse tarayici local storage temizlenmeli veya uygulamaya taslak sifirlama butonu eklenmelidir.
