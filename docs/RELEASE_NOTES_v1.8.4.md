# v1.8.4 Release Notes

## Paket hedefi

Yeni Fis ekraninda sorun devam ederse kullanicinin aktif taslagi sifirlayabilmesini saglamak.

- Ayarlar ekranina Aktif Taslagi Sifirla aksiyonu eklendi.
- Guvenlik aciklamasi Yeni Fis temiz baslangic bilgisini icerecek sekilde guncellendi.
- Aktif taslak sifirlama dokumani eklendi.
- Paket surumu 1.8.4 olarak hizalandi.
