# v1.3.7 APK Decision Matrix

## Amac

APK build baslatma kararini daha net vermek.

## APK build baslatilabilir

- Web Preview guncel ve yesilse
- Demo turu tek oturumda tamamlanabiliyorsa
- Yeni Fis barkod akisi webde sorun cikarmiyorsa
- QR Album fiyat gostermiyorsa
- APK oncesi check komutlari lokal ortamda geciyorsa

## APK build ertelenmeli

- Web Preview eski surum gosteriyorsa
- Ekranlarda ciddi tasma varsa
- Yeni Fis akisi tamamlanamiyorsa
- QR Album veya Mesajlar ekrani acilmiyorsa
- Honeywell barkod focus davranisi belirsizse

## Karar

APK build, demo turu ve lokal check sonrasinda ayrica onayla baslatilmalidir.
