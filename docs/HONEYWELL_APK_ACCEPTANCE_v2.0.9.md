# v2.0.9 Honeywell APK Acceptance

## Amac

APK ve Honeywell test fazi icin kabul kriterlerini netlestirmek.

## Kabul kriterleri

- Uygulama Honeywell cihazda acilir.
- Login ekrani tasma yapmaz.
- Dashboard acilir.
- Yeni Fis ekrani acilir.
- Barkod input focus alir.
- Scanner urun kodunu inputa aktarir.
- Fiyat karti gorunur.
- Adet alani kullanilabilir.
- Urun fise eklenir.
- QR Album fiyat gostermez.
- Geri tusu beklenen sekilde calisir.

## Karar

Bu kriterler gecmeden APK canli kullanim adayi sayilmaz.
