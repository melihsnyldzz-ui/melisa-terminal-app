# v2.0.1 Web Preview Validation

## Amac

Web Preview uzerinden Yeni Fis fiyat/adet akisini dogrulamak.

## Kontrol sirasi

1. Login yap.
2. Dashboard uzerinden Yeni Fis ekranina gir.
3. Musteri sec.
4. Fisi baslat.
5. Demo kod ile urun bul.
6. Fiyat kartini kontrol et.
7. Adet gir.
8. Urunu fise ekle.
9. Fis satirinda birim fiyat ve satir tutarini kontrol et.
10. QR Album ekraninda fiyat gorunmedigini kontrol et.

## Kabul kriteri

Yeni Fis fiyatli operasyon ekrani gibi calisir. QR Album fiyatsiz musteri vitrini olarak kalir.
