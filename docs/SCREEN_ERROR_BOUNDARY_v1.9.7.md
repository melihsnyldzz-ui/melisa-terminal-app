# v1.9.7 Screen Error Boundary

## Amac

Bir ekran runtime hata verirse uygulamanin tamamen beyaz ekranda kalmasini engellemek.

## Gelen davranis

- Ekran hata verirse guvenli hata karti gosterilir.
- Hangi ekranin acilamadigi kullaniciya gosterilir.
- Dashboard'a Don aksiyonu sunulur.
- Kullanici Dashboard'a donup Ayarlar ekranindan Aktif Taslagi Sifirla aksiyonunu kullanabilir.

## Neden onemli

Yeni Fis gibi kritik ekranlarda hata varsa kullanici tamamen kilitlenmez.
