# v1.7.0 ERP Price Read Boundary

## Amac

Gercek ERP entegrasyonunda fiyat bilgisinin nasil okunacagini sinirlamak.

## Ilk faz

- Sadece fiyat okuma hedeflenir.
- Canli veri yazma yoktur.
- Fis kaydi canli sisteme yazilmaz.
- Urun fiyat bilgisi barkodla okunur.
- Fiyat personel ekraninda gorunur.

## Yazma islemleri

Canli fis yazma, stok dusme veya ERP kaydi ayri onay fazinda ele alinmalidir.

## Risk

Fiyat okuma dogru calismadan canli satis yazma fazina gecilmemelidir.
