# v1.0.7 Web Preview Demo Route

## Demo rotasi

1. Login ekraninda PIN gir ve giris yap.
2. Dashboard uzerinden Yeni Fis ekranina gec.
3. Musteri sec ve fisi baslat.
4. Hizi demo kodlardan urun ekle.
5. QR Album Hazirla aksiyonunu kullan.
6. Acik Fisler ekraninda arama ve filtreleri dene.
7. Mesajlar ekraninda fis no veya mesaj basligi ara.
8. Gonderilemeyenler ekraninda tekrar deneme akisini kontrol et.
9. Veri Guncelle ekraninda mock senkronu calistir.
10. Ayarlar ekraninda terminal sagligi kartlarini kontrol et.

## Amac

Web preview uzerinden uygulamanin temel mock operasyon akisini tek turda kontrol etmek.
