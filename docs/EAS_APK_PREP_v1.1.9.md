# v1.1.9 EAS APK Prep Notes

## Amac

Web Preview demo asamasi sonrasinda APK hazirlik surecini netlestirmek.

## APK oncesi kontrol

- Web Preview yesil deploy olmali.
- Login ekraninda guncel surum gorunmeli.
- Yeni Fis demo kodlari calismali.
- QR Album fiyat gostermeden acilmali.
- Mesajlar ve Acik Fisler arama akislari calismali.
- Ayarlar terminal sagligi kartlari gorunmeli.

## EAS hazirlik karari

APK build baslatilmadan once Honeywell veya Expo preview ile ekranlar kontrol edilmelidir.

## APK sonrasi kontrol

- Uygulama adi dogru mu?
- Ekran tasmasi var mi?
- Barkod input focus aliyor mu?
- Android geri tusu dogru calisiyor mu?
- Demo fis akisi tamamlanabiliyor mu?
