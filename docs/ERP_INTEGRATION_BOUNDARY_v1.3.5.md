# v1.3.5 ERP Integration Boundary

## Amac

Gercek ERP entegrasyonuna gecmeden once sinirlari netlestirmek.

## Bu asamada yapilmayanlar

- Canli SQL baglantisi yok
- Canli stok sorgusu yok
- Canli musteri karti cekme yok
- Canli fis kaydi yok
- Canli kullanici yetkilendirme yok

## ERP oncesi hazirlik

- Mock akisin Web Preview uzerinden onaylanmasi
- Honeywell cihaz testinin tamamlanmasi
- Hangi Vega/ERP endpointlerinin kullanilacaginin belirlenmesi
- Baglanti ve yetki modelinin netlesmesi
- Offline kuyruk davranisinin onaylanmasi

## Risk notu

ERP entegrasyonu baslamadan once veri yazma islemleri ayrica guvenlik onayi gerektirir.
