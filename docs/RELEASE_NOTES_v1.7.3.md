# v1.7.3 Release Notes

## Paket hedefi

Yeni Fis ekraninin acilmamasina neden olabilecek eski taslak fiyat uyumsuzlugunu duzeltmek.

## Degisiklikler

- Aktif satis taslagi yuklenirken satirlar normalize edilir.
- Fiyati olmayan eski taslak satirlarinda fiyat 0 olarak atanir.
- Adet degeri bos veya gecersizse 1 olarak atanir.
- Yeni Fis ekraninin eski taslaklardan dolayi beyaz ekrana dusme riski azaltildi.

## Not

Bu bugfix, fiyat/adet akisi sonrasi eski local taslaklarla uyumluluk icindir.
