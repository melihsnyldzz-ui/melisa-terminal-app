# v2.1.5 Honeywell Install Test

## Amac

APK kurulduktan sonra Honeywell cihazda ilk kurulum testini yapmak.

## Kurulum sonrasi kontrol

1. Uygulama aciliyor mu?
2. Login ekrani tasma yapmadan gorunuyor mu?
3. Dashboard aciliyor mu?
4. Yeni Fis ekrani aciliyor mu?
5. Musteri secimi calisiyor mu?
6. Barkod input focus aliyor mu?
7. Scanner okutmasi inputa dusuyor mu?
8. Fiyat karti gorunuyor mu?
9. Adet girisi yapilabiliyor mu?
10. Urun fise ekleniyor mu?
11. QR Album fiyat gostermeden aciliyor mu?
12. Android geri tusu beklenen sekilde calisiyor mu?

## Karar

Bu test gecmeden APK canli kullanim adayi sayilmaz.
