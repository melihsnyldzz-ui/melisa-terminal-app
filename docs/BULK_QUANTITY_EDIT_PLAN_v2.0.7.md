# v2.0.7 Bulk Quantity Edit Plan

## Amac

Yeni Fis ekraninda ileriki fazda toplu adet duzenleme ihtiyacini planlamak.

## Mevcut durum

- Urun okutulur.
- Fiyat gorulur.
- Adet girilir.
- Urun fise eklenir.
- Satirda artir/azalt butonlari vardir.

## Ileriki hedef

- Fis satirinda adete direkt dokunup sayi girilebilmesi.
- Yanlis adet girildiyse satirin tamamen silinmeden duzeltilebilmesi.
- Toplam tutarin anlik guncellenmesi.

## Kabul kriteri

Adet duzenleme satis hizini yavaslatmamalidir.
