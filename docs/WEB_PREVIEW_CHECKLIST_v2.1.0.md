# v2.1.0 Web Preview Checklist

## Amac

Web Preview uzerinden demo akisini ve fiyat/adet davranisini tekrar kontrol etmek.

## Kontrol sirasi

1. Login yap.
2. Dashboard aciliyor mu kontrol et.
3. Yeni Fis ekranina gir.
4. Musteri sec.
5. Fis baslat.
6. Demo urun kodu ile urun bul.
7. Fiyat gorunuyor mu kontrol et.
8. Adet gir.
9. Urunu fise ekle.
10. Satir tutari ve toplam tutari kontrol et.
11. QR Album ekraninda fiyat gorunmedigini kontrol et.
12. Ayarlar ekraninda Aktif Taslagi Sifirla aksiyonunu kontrol et.

## Kabul kriteri

Demo turu tek oturumda tamamlanmalidir.
