# v1.4.7 Web Preview Checklist

## Amac

Web Preview uzerinden hizli demo ve ekran kontrolunu standart hale getirmek.

## Kontrol sirasi

1. Web Preview linkini gizli sekmede ac.
2. Login ekraninda guncel surum etiketini kontrol et.
3. PIN girerek Dashboard ekranina gec.
4. Demo Test Rotasi kartini kontrol et.
5. Yeni Fis ekraninda musteri sec ve urun ekle.
6. QR Album ekraninda urun listesi ve WhatsApp onizlemeyi kontrol et.
7. Acik Fisler ekraninda arama ve filtreleri dene.
8. Mesajlar ekraninda arama yap.
9. Gonderilemeyenler ekraninda tekrar dene akisini dene.
10. Veri Guncelle ve Ayarlar ekranlarini ac.

## Kabul kriteri

- Ekranlar acilmali.
- Kritik butonlar calismali.
- Fiyat bilgisi gorunmemeli.
- Demo turu tek oturumda tamamlanmali.
- Ekranlarda ciddi tasma olmamali.
