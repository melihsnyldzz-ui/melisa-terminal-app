# v1.9.1 Web Preview Clean Start Guide

## Amac

Yeni Fis ekraninda beyaz ekran veya eski veri sorunu gorulurse temiz baslangic adimlarini netlestirmek.

## Temiz baslangic sirasi

1. Web Preview linkini gizli sekmede ac.
2. Login yap.
3. Ayarlar ekranina gir.
4. Aktif Taslagi Sifirla aksiyonunu calistir.
5. Dashboard ekranina don.
6. Yeni Fis ekranina tekrar gir.
7. Musteri sec ve fisi baslat.
8. Demo urun kodu ile fiyat/adet akisini dene.

## Hala beyaz ekran olursa

- Tarayici cache temizlenmeli.
- Farkli gizli sekme denenmeli.
- Local build veya cihaz logu alinmalidir.

## Not

Ayarlar ekranindaki Aktif Taslagi Sifirla aksiyonu terminal ayarlarini veya oturumu silmez.
