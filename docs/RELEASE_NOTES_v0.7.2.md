# v0.7.2 Release Notes

## Paket hedefi

Bu paket, terminal uygulamasında v0.6.8 sonrası yarım kalan hızlı paket kapanışını tamamlamak için hazırlandı.

## Gelen değişiklikler

- Dashboard son aktif fiş kartı güçlendirildi.
- Aktif fiş kartına QR Albüm ve Açık Fişler kısayolları eklendi.
- Gönderilemeyenler ekranında hata nedeni daha görünür hale getirildi.
- Gönderilemeyenler ekranına belge ve işlem tipi özeti eklendi.
- Paketli GitHub çalışma modeli sürdürüldü.

## Sürüm

- package.json version: 0.7.2
- app.json expo.version: 0.7.2

## Not

Bu paket GitHub üzerinden uygulanmıştır. Local build, Expo preview ve gerçek Honeywell cihaz testi çalıştırılmadı.
