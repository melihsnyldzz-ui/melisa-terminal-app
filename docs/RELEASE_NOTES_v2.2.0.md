# v2.2.0 Release Notes

## Paket hedefi

APK ve Honeywell test fazina gecis icin dosya ve dokuman hazirligini tamamlamak.

## Degisiklikler

- EAS profil inceleme notu eklendi.
- APK build komut rehberi eklendi.
- Honeywell kurulum test notu eklendi.
- Paket surumu 2.2.0 olarak hizalandi.

## APK build komutu

```powershell
npx eas build --platform android --profile preview
```

## Not

Build bu paket icinde calistirilmadi. Local check ve onay sonrasi baslatilmalidir.
