# v1.7.2 Release Notes

## Paket hedefi

Yeni Fis fiyat ve direkt adet girme akisini test edilebilir hale getirmek.

## Degisiklikler

- Yeni Fis fiyat ve adet test checklist eklendi.
- Fiyat gorunurluk kurali eklendi.
- Honeywell fiyat ve adet test notlari eklendi.
- Web Preview fiyat ve adet kontrol notlari eklendi.
- APK fiyat ve adet kabul kriterleri eklendi.
- ERP fiyat okuma siniri eklendi.

## Ana kural

Fiyat Yeni Fis ekraninda gorunur. QR Album ekraninda fiyat gorunmez.

## Not

Bu paket, v1.6.4 ile gelen fiyat ve direkt adet akisini test ve kabul kriterleriyle tamamlar.
