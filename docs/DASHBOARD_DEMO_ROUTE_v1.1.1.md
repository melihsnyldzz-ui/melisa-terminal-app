# v1.1.1 Dashboard Demo Route

## Amac

Dashboard ekranindan baslayan hizli demo turunu netlestirmek.

## Demo rotasi

1. Dashboard uzerinden Yeni Fis ekranina gec.
2. Musteri sec ve urun ekle.
3. Dashboard'a geri don.
4. Son aktif fis kartini kontrol et.
5. QR Album kisayolunu kullan.
6. Acik Fisler kisayolunu kullan.
7. Mesajlar ve Gonderilemeyenler modullerini kontrol et.
8. Veri Guncelle ve Ayarlar ekranlarini ac.

## Beklenen sonuc

Dashboard ana kontrol paneli gibi calisir. Kullanici temel demo turunu buradan baslatip tamamlayabilir.
